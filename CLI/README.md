# CLI(Command Line Interface)
it is a text-based tools used for exploring, debugging, and interacting with various elements of a ROS system like nodes, topics, services....


# Structure
```
ros2 <command> <verb> [options]
```
1. Command
- command is a large category to decide what kind of domain you will use

2. Verb
- verb is a deatil category to decide what function you will use in that domain

3. API
- Interface part that is compatitible to command/verb
